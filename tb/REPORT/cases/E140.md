## Intent

- **Scenario:** `csr_counter_freeze` atomicity at the freeze edge: write CSR2 bit5=1 in the exact cycle a `push_write_grant` lands
- **Primary checks:** the snapshot `debug_msg2_snap.push_cnt` at the freeze edge equals the counter value as seen by the always_ff block on that posedge (i.e. the value before the same-cycle increment, since the CSR write block reads the current `debug_msg2` register); subsequent CSR reads of word 6 / word 11 return that exact value while bit5=1; catches any read-after-write race in the freeze copy logic
