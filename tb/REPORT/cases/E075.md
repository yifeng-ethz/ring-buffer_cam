# ❓ E075

**Bucket:** `EDGE` &nbsp; **Method:** `D` &nbsp; **Build:** `default_p2_pipe4` &nbsp; **Effort:** `smoke` &nbsp; **Result:** `pending`

## Intent

- **Scenario:** overwrite exactly at boundary between partition 3 and partition 0: last write slot 511 then wrap slot 0
- **Primary checks:** slot 511 is P3 bit 127, slot 0 is P0 bit 0; overwrite on slot 0 must update P0's snapshot, not P3's; catches partition index computed from `cam_wr_addr` vs from `write_pointer` mismatch
